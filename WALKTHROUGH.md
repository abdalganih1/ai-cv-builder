# AI CV Builder - Walkthrough
## نظرة عامة
تم بناء تطبيق ويب متكامل لإنشاء السير الذاتية باستخدام Next.js 14, Tailwind CSS, و TypeScript. التطبيق يتميز بواجهة مستخدم حديثة وتجربة تفاعلية تعتمد على الذكاء الاصطناعي.

## المميزات الأساسية
### 1. واجهة المستخدم (UI/UX)
- تصميم "خطوات متتالية" (Wizard) مع شريط تقدم ذكي.
- ألوان احترافية: **Professional Blue** (#1e3a5f) و **Teal** (#0891b2).
- تأثيرات حركية (Animations) ناعمة باستخدام `framer-motion`.

### 2. محرك الاستبيان الذكي (AI Questionnaire)
- **ملف:** `src/lib/ai/questionnaire-agent.ts`
- **الآلية:**
    - يحلل طول إجابات المستخدم.
    - يولد أسئلة إضافية إذا كانت الإجابات مختصرة أو مفصلة.
    - يقترح مهارات بناءً على الوظيفة المستهدفة.

### 3. محاكاة الدفع (Sham Cash)
- **المكون:** `src/components/payment/ShamCashPayment.tsx`
- واجهة تحاكي عملية الدفع عبر "شام كاش".
- تكلفة ثابتة: 500 ل.س.

### 4. المعاينة المباشرة والتعديل (Chat-to-Edit)
- **المكون:** `src/components/preview/CVPreview.tsx`
- **الميزة:** معاينة حية للسيرة الذاتية مع واجهة دردشة جانبية.
- يمكن للمستخدم طلب تعديلات بلغة طبيعية (مثال: "اجعل النبذة أكثر اختصاراً")، ويقوم `src/lib/ai/chat-editor.ts` بمعالجة الطلب وتحديث البيانات فوراً.

## كيفية التشغيل
1. تأكد من إضافة مفتاح API في ملف `.env.local`:
   ```bash
   ZAI_API_KEY=your_key_here
   ```
2. تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
3. فتح المتصفح على `http://localhost:3000`

## لقطات (Hypothetical)
- **شاشة الترحيب:** تطلب الاسم مع تصميم نظيف.
- **الاستبيان:** أسئلة تظهر تباعاً بناءً على الإجابات السابقة.
- **الدفع:** زر محاكاة الدفع مع حالة التحميل.
- **المعاينة:** السيرة الذاتية على اليسار والدردشة على اليمين.

## الهيكلية التقنية
- **Next.js App Router**: لإدارة المسارات.
- **State Management**: استخدام `useState` في المكون الجذر `page.tsx`.
- **Styling**: Tailwind CSS مع متغيرات CSS مخصصة للألوان.
